### SecretSanta
- новогодний рандомайзер для внутреннего пользования компании.

##### Стек технологий:

* Front: JavaScript, HBS, HTML, CSS;
* Back: Node.js, Express, Sessions, Bcrypt;
* DB: PostgreSQL, Sequelize ORM.

##### Установка:

* npm ci
* npx sequelize db:create
* npx sequelize db:migrate
* npm run dev
